# Aplikasi-Pemilihan-Buku-Berdasarkan-Genre
Program "Aplikasi Pemilihan Buku Berdasarkan Genre" adalah aplikasi berbasis Python yang memungkinkan pengguna mencari rekomendasi buku sesuai preferensi genre mereka. Program ini memuat data buku (JSON atau CSV) dan menampilkan genre yang tersedia. Pengguna dapat kembali memilih genre lain untuk menemukan lebih banyakÂ rekomendasi.
